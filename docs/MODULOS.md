# Catálogo de módulos

| Módulo | Función principal | Estado en 2.0 |
|---|---|---|
| Resumen | KPI, actividad y accesos rápidos. | Vue 3 nativo. |
| Mi finca | Datos generales de la unidad productiva. | Adaptador funcional; pendiente SFC. |
| Lotes | Parcelas, variedades, área y estado. | Adaptador funcional; pendiente SFC. |
| Cosecha | Cortes, recolectores, madurez y pagos. | Adaptador funcional; pendiente SFC. |
| Beneficio | Fermentación, secado y rendimiento. | Adaptador funcional; pendiente SFC. |
| Inventario | Entradas, salidas, venta y existencia. | Adaptador funcional; pendiente SFC. |
| Gastos | Costos por categoría, lote y período. | Adaptador funcional; pendiente SFC. |
| Reportes | Producción, costos y rentabilidad. | Adaptador funcional; pendiente SFC. |
| Sostenibilidad | Emisiones, prácticas y certificaciones. | Adaptador funcional; pendiente SFC. |
| Calidad | Cataciones, atributos y puntaje SCA. | Vista funcional añadida; pendiente SFC. |
| Trazabilidad | Códigos de origen, QR y cadena hash. | Adaptador funcional; pendiente SFC. |
| Predictivo | Estimaciones y análisis agronómico. | Adaptador funcional; pendiente SFC. |
| Mercado | Precios y benchmarks. | Adaptador funcional; datos demo marcados como sintéticos. |
| Marketing | Clientes, campañas y fidelización. | Adaptador funcional; pendiente SFC. |
| Clima | Historial meteorológico y alertas. | Adaptador funcional; pendiente SFC. |
| Perfiles de café | Sabores y recomendaciones por cliente. | Adaptador funcional; pendiente SFC. |
| Educación | Artículos y tips contextuales. | Adaptador funcional; pendiente SFC. |
| Ayuda | Manual integrado y soporte. | Adaptador funcional; pendiente SFC. |

“Adaptador funcional” significa que el módulo original se monta dentro del shell Vue 3 y conserva sus operaciones. No significa que su DOM interno ya haya sido reescrito como componente Vue.
