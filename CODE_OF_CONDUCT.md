# Código de conducta

Esta comunidad busca construir tecnología útil para productores, técnicos, estudiantes y organizaciones cafetaleras.

Se espera comunicación respetuosa, crítica técnica sustentada, apertura a personas con distintos niveles de experiencia y protección de información real de fincas o clientes. No se toleran ataques personales, discriminación, acoso, publicación de datos privados ni sabotaje deliberado.

Las decisiones deben priorizar seguridad de datos, claridad para el usuario, sostenibilidad del proyecto y beneficio verificable para el sector cafetalero.
