# Soporte

## Ayuda comunitaria

- Documentación del repositorio.
- GitHub Discussions para preguntas y casos de uso.
- GitHub Issues para errores reproducibles y solicitudes de mejora.

## Qué debe incluir una solicitud

- Versión de Cafetal OS.
- Sistema operativo.
- Pasos realizados.
- Resultado esperado y resultado obtenido.
- Capturas o logs sin información privada.

## Servicios profesionales

NaT Lab puede desarrollar en el futuro servicios de instalación, capacitación, migración, personalización e implementación. Dichos servicios serán independientes de los derechos concedidos por la licencia open source.
