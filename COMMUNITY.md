# Comunidad Cafetal OS

Cafetal OS quiere conectar conocimiento de campo y conocimiento tecnológico.

## Espacios de conversación

Mientras se habilitan canales oficiales, utilice:

- Issues para errores y propuestas concretas.
- Discussions para preguntas, ideas y experiencias de uso.
- Pull requests para cambios revisables.

No utilice los issues para publicar datos de una finca real.

## Cómo formular una buena conversación

- Explique el contexto sin asumir que todos conocen el proceso.
- Use términos locales y aclare equivalencias.
- Diferencie observación, requisito y solución propuesta.
- Comparta ejemplos ficticios.
- Indique región o tipo de operación cuando cambie el flujo, sin revelar ubicación exacta.

## Temas prioritarios

- Productividad y costos por lote.
- Calidad, perfiles y trazabilidad.
- Manejo de cosecha y beneficio.
- Sostenibilidad y cumplimiento.
- Experiencia offline y equipos de bajos recursos.
- Capacitación de usuarios no técnicos.
- Interoperabilidad con otras herramientas abiertas.

## Reuniones comunitarias

Una reunión debe tener agenda, notas públicas y decisiones registradas. Las demostraciones deben usar la base demo.
